hexo总结

命令

  • hexo g 生成html文件
  • hexo clean 清除html文件、缓存
  • hexo s 打开server，可以在本地预览 http://localhost:4000/
  • hexo d 将修改上传到github上，配和hexo g 使用
  • hexo n xxxx 新建一个post，xxxx是post的名字

数学公式

参考这篇文章可以解决大多数数学公式的问题

如果使用Typora，那么，文件->偏好设置-> markdown ->内联公式(打勾),即可用 $$ 表示行内公式。
